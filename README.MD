# VK Registration And Auth API 2024

## Requirement

- PHP 8.2
- MySql DBMS
- Composer

## Installation

1. Run `composer install`
2. Run `php ./marusia migrate`

## Running

1. Run `php -S localhost:8080 -t public`
2. Visit http://localhost:8080
3. Enjoy!

## Extra

- You can run `php ./marusia empty <tablename>` to truncate table.
